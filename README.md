Upload Avatar From URL for XenForo 2.1.10.2+
============================================

Description
-----------

This add-on allows users with permission to upload avatar from URL.

Requirements
------------

- PHP 7.3+

Permissions
-----------

| Group               | Permission                |
| ------------------- | ------------------------- |
| General permissions | Upload an avatar from URL |
